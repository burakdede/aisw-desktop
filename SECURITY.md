# Security Policy

## Supported scope

Please report vulnerabilities in the desktop application, the Tauri command layer, local storage handling, updater configuration, and bundled runtime integration.

## Reporting

Do not open a public GitHub issue for a suspected security problem.

Report the issue privately to the project maintainer with:

- a clear description of the issue
- impact and affected platforms
- reproduction steps or a proof of concept
- any suggested mitigation if available

## Handling expectations

- Reports will be acknowledged as quickly as practical.
- Sensitive details should stay private until a fix or mitigation is available.
- Please avoid exfiltrating real credentials or destructive testing against machines you do not own.
