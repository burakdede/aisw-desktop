# Code of Conduct

## Our standard

This project expects respectful, professional collaboration. Be clear, direct, and constructive. Harassment, discrimination, intimidation, and bad-faith participation are not acceptable.

## Expected behavior

- Assume good intent and discuss disagreements on technical merits.
- Give actionable feedback and keep review comments specific.
- Respect user privacy, credentials, and local-only data.
- Accept correction quickly when behavior or code causes harm.

## Unacceptable behavior

- Personal attacks, insults, or harassment.
- Publishing private information without permission.
- Disruptive behavior that blocks maintainers or contributors from working productively.

## Enforcement

Project maintainers may edit, remove, or reject contributions that violate this code of conduct. Repeated or severe violations can result in temporary or permanent bans from the project spaces.
